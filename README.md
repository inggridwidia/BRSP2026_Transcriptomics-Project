This repository showcases a collection of bioinformatics projects developed during the **Bioinformatics Research Starter Program 2026** by **OmicsLite**. The primary focus is the application of transcriptomics and Differential Gene Expression (DGE) analysis across various medical case studies.

## 📂 Repository Structure
This repository follows a comprehensive bioinformatics workflow, from raw data acquisition to functional pathway interpretation:

1. [01-GEO2R-Intestinal-RYGB](https://github.com/inggridwidia/BRSP2026_Transcriptomics-Project/tree/main/01-GEO2R-Intestinal-RYGB): Initial exploratory analysis using the GEO2R tool to identify transcriptomic shifts in intestinal tissue following Roux-en-Y Gastric Bypass (RYGB) surgery.

2. [02-Neurovascular-DM2-Analysis](https://github.com/inggridwidia/BRSP2026_Transcriptomics-Project/tree/main/02-Neurovascular-DM2-Analysis): A detailed study of gene expression profiles in the human brain's neurovascular unit under Type 2 Diabetes Mellitus (GSE161355). This project successfully identified 77 key genes (Common DEGs) exhibiting synchronous functional failure across both endothelial cells and neurons.

3. [03-Capstone-Transcriptomics-RYGB-Diabetes](https://github.com/inggridwidia/BRSP2026_Transcriptomics-Project/tree/main/03-Capstone-Transcriptomics-RYGB-Diabetes): A Capstone project integrating advanced data analysis skills in R (`limma`, `clusterProfiler`) to map metabolic adaptations and clinical recovery mechanisms in T2DM patients.

## 🛠️ Technical Skills
* **Programming:** R (BioConductor)
* **Statistical Analysis:** RMA Normalization, Linear Modeling (limma), and False Discovery Rate (FDR) correction.
* **Data Visualization:** Volcano Plots, Hierarchical Clustering (Heatmaps), and Venn Diagrams.
* **Pathway Analysis:** Gene Ontology (GO) & KEGG Pathway Enrichment Analysis.
